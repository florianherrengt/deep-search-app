# Keyboard Shortcuts

## App

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + N` | New chat |
| `Cmd/Ctrl + ,` | Open preferences |

## Composer (chat input)

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift + Enter` | New line |
| `Tab` | Cycle forward through previous search queries |
| `Shift + Tab` | Cycle backward through previous search queries |

## Research sidebar

| Shortcut | Action |
|----------|--------|
| `Ctrl + Tab` | Next search folder |
| `Ctrl + Shift + Tab` | Previous search folder |
| `Enter` | Submit search |
| `Escape` | Clear search input |

## Research sidebar cycling

`Ctrl + Tab` cycles through your previous searches in the sidebar. The cycling wraps around — pressing `Ctrl + Tab` on the last search jumps to the first, and `Ctrl + Shift + Tab` on the first jumps to the last.

This shortcut works regardless of where your focus is, including inside the chat composer. The composer's own `Tab` key behavior (cycling previous search queries) is unaffected — plain `Tab` and `Shift + Tab` continue to work for query history.
